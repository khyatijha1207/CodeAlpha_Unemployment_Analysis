# Unemployment Analysis in India using Python

## Project Overview

This project analyzes unemployment trends in India using Python. The dataset contains unemployment rates, employment figures, labour participation rates, and area-wise information.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Objectives

* Analyze unemployment trends in India
* Study the impact of Covid-19 on unemployment
* Compare Urban and Rural unemployment rates
* Identify states with the highest unemployment rates
* Generate insights through data visualization

## Visualizations

1. Unemployment Rate Over Time
2. Top 10 States with Highest Unemployment
3. Urban vs Rural Unemployment
4. Covid-19 Impact Analysis
5. Labour Participation Rate Distribution
6. Correlation Heatmap

## Key Insights

* Covid-19 significantly increased unemployment rates.
* Some states consistently recorded higher unemployment.
* Urban and rural regions showed different unemployment patterns.
* Employment and labour participation indicators are strongly related.

## Conclusion

The analysis highlights the employment challenges faced during the Covid-19 period and provides useful insights for economic planning and policy-making.
